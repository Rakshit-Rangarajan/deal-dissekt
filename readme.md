# Deal Dissekt 📈

[![Angular](https://img.shields.io/badge/Angular-21.x-DD0031?style=flat-square&logo=angular)](https://angular.dev)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.x-3178C6?style=flat-square&logo=typescript)](https://www.typescriptlang.org/)

**Deal Dissekt** is a premium, developer-centric platform built for **Abhishek Venkatesh**, a CFA Charterholder and Chartered Accountant specializing in M&A and Private Equity Advisory. The platform is designed to provide high-fidelity financial insights, dissecting complex deal structures, market trends, and regulatory updates with a sleek, minimalist aesthetic.

---

## ✨ Features

- **🚀 Immersive Loading Experience**: A skippable, high-performance loading screen with the brand logo and smooth grid-layer transitions.
- **📑 Dynamic insights Engine**: An advanced blog system with category-based filtering for multiple financial domains:
  - **Deal Insights**: Deep dives into Cross-Border M&A and LBO value creation.
  - **DStreet Dissekt**: Real-time analysis of Dalal Street (Indian Market) trends.
  - **Regulatory Updates**: Simplified breakdowns of SEBI circulars and RBI policies.
  - **Tax Dissekt**: Professional advisory on GAAR, ESOP taxation, and corporate restructuring.
  - **Startup Insights**: Analysis of FinTech pivots and VC due diligence.
- **🎨 Premium UI/UX**:
  - Dark/Light mode engine for optimal readability.
  - Glassmorphic components and vibrant custom gradients.
  - Custom scroll animations integrated via `ScrollService`.
- **⚡ Modern Tech Stack**: Built with Angular 21, leveraging **Angular Signals** for reactive state management and lazy loading for maximum performance.

---

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS)
- Angular CLI

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Rakshit-Rangarajan/deal-dissekt.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm start
   ```
   Navigate to `http://localhost:4200/`.
